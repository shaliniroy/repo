Objective
=========
To design a package which we can install in our system for shell with commands stock and greet.

Download
--------
`click here <https://testpypi.python.org/packages/source/s/shellproject/shellproject-1.tar.gz#md5=7c60f8ef86d7eaf38ba92cfb77b922b8>`_ to download the package

How to Install
--------------
Read the README file
